# Goals Framework — General, Company-Specific, Department, Employee

## Hierarchy of Goals

1. **General Goals** → Company-wide, all employees, 2026 vision
2. **Company-Specific Goals** → Per company (BejCapital, Brentford, Altair, Rockerforce)

4. **Employee-Specific Goals** → Per role, measurable, weekly/daily
